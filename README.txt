============================
=== Requisitos
============================
O projecto está pré-compilado na pasta Release.
Pode ser compilado através do Visual Studio 2010 com ligação ao ARToolkit

O programa tem as seguintes dependências (incluidas na directoria Release):
DSVL.dll, glut32.dll, libARvideo.dll, MSVCP71.dll, msvcr71.dll

Estas duas últimas podem não funcionar em determinadas instalações (testado em Windows 7 x86)
Se for o caso, a solução deverá passar por instalar o Microsoft Visual C++ Redistributable
